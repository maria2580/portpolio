핵심 백엔드 개발 역량

Java & Spring Boot

RESTful API 서버 설계 및 개발, JPA를 통한 DB 연동, 기본 트랜잭션 처리에 능숙합니다.

주요 프로젝트: '정보 보안 공모전 백엔드', 'Smart Factory Server', 'Today_menu_DB_api'

학습 목표: Spring Security 심화, 대용량 트랜잭션 처리, MSA 환경에서의 Spring Cloud 활용, 비동기 처리.

데이터베이스 (MySQL & JPA)

ERD 설계, SQL 작성, JPA를 통한 ORM 활용 및 MySQL 데이터베이스 운영 경험을 보유하고 있습니다.

주요 프로젝트: 다수 프로젝트에서 데이터 모델링 및 관리

학습 목표: 쿼리 최적화, 고급 인덱싱, NoSQL(Redis, MongoDB) 활용.

API 설계 & 문서화

RESTful API 원칙 기반 설계, Swagger를 사용한 효과적인 API 문서화 경험이 있습니다.

주요 프로젝트: '정보 보안 공모전' (Swagger API 문서화)
